# Micron Bush Manufacturers (MBM) – Website

A modern industrial manufacturing website for **Micron Bush Manufacturers** built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS** – styling & design system
- **Framer Motion** – scroll and hover animations
- **Lucide React** – icons
- **React Router** – routing

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview   # preview production build
```

## Structure

- `src/components/` – Navbar, Hero, About, Products, Why Choose MBM, Customers slider, Footer
- `src/pages/` – Home, About Us
- `src/data/products.js` – product/service list
- Color palette: `#fbfbfc`, `#0662a3`, `#979a9f`
